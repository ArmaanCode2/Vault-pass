BUILD A COMPLETE ANDROID PASSWORD MANAGER APPLICATION
Project Name

VaultPass

Project Objective

Create a complete production-ready Android application that functions as a secure offline password manager.

The application must be capable of being compiled into a downloadable APK and installed directly on Android devices.

This application is intended for personal password storage only.

All password data must remain on the device and must never be transmitted to any server, cloud service, analytics provider, telemetry service, third-party API, or external database.

The application must function completely offline after installation.
